Base Building

A Hardcore base is a survival machine. Every room should either hide you, feed you, or help you escape.

Core layout priorities

  • Store critical items in separated vaults, not one main chest wall.
  • Create two independent exits with different route types.
  • Design bunker rooms below Y=28 for stealth and fallback value.
  • Install hidden emergency kits near each route checkpoint.

Raid-resistant habits

  • Never reveal your true loot room in first tour or first trust test.
  • Use decoy furnaces, benches, and fake supply paths.
  • Keep farms distributed so one breach does not wipe production.

Infrastructure with misc extras

  • Build internal highways from approved materials for faster response: Path Speed Materials.
  • Use team proximity communication for live breach callouts: Simple Voice Chat Integration.
  • Patrol rooftops at night with torch carry behavior in mind: Torch & Phantom Control.
