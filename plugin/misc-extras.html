Purpur Misc Extras

Quick reference for currently enabled Purpur extras only. Disabled toggles are intentionally omitted.

Nether Build Height Guard

  • Enabled: true
  • Height limit: 256
  • Message: Max build height in this world is: 256

Sleep Progress Chat Messages

  • Enabled: true
  • Player sleeping: A player has fallen asleep. Sleeping progress is shown for World.
  • Skipping night: Enough players have slept! Skipping through the night in World.

Mob / Block QoL

  • Snow golems drop pumpkins when sheared
  • Shift-right-click can place invisible item frames

Gameplay Rules

  • Respawn anchors require charges
  • Fall damage applies when jump boost is active
  • Totems work on void death
  • Raid totem drops enabled (25% chance)

Run Faster on Path Blocks

  • Enabled: true
  • Modifier type: add_scalar
  • Speed value: 0.25
  • Path blocks: DIRT_PATH, SMOOTH_STONE, END_STONE_BRICKS, END_STONE_BRICK_STAIRS, PURPUR_BLOCK, PURPUR_STAIRS, POLISHED_BLACKSTONE_STAIRS, POLISHED_BLACKSTONE_BRICKS, PRISMARINE_BRICKS, PRISMARINE_BRICK_STAIRS, SMOOTH_RED_SANDSTONE, RESIN_BRICKS, RESIN_BRICK_STAIRS, TUFF_BRICKS, TUFF_BRICK_STAIRS, DEEPSLATE_TILES, DEEPSLATE_TILE_STAIRS, DEEPSLATE_BRICKS, DEEPSLATE_BRICK_STAIRS, POLISHED_DEEPSLATE, POLISHED_DEEPSLATE_STAIRS, STONE_BRICKS, STONE_BRICK_STAIRS

Anvil Extras

  • Anvils can split minecarts
  • Anvils can split boats
  • Anvil crushes blocks: cobblestone → gravel, gravel → sand

Dispenser Extras

  • Can break blocks with: pickaxe, shovel, hoe, axe, shears
  • Can insert music discs into jukeboxes
  • Can interact with cauldrons
